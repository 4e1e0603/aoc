e.g.

### kdb+/q

    C:\q\w64\q.exe
    q)system "cd C:/path/to/advent-of-code"

    q)\l aoc{yy}{dd}.q

e.g.

    q)\l aoc2201.q

## Template

Use PowerShell
- download data from `https://adventofcode.com/{yyyy}/day/{d}/input`
- create file with proper name, defaults and comment `aoc{yy}{dd}.{ex}`
